# DEV Intelligence Factory Lab

Laboratorio local para demostrar cómo una empresa puede pasar del uso individual de IA a una capacidad corporativa gobernada, segura, trazable y medible en entornos DEV.

## Principio clave

La IA propone, el pipeline valida y el equipo técnico decide.

## Restricciones

- Solo DEV.
- Sin producción.
- Sin secretos reales.
- Sin terraform apply autónomo en empresa real.
- Sin merge directo.
- Revisión humana obligatoria.
- Pipeline obligatorio.

## Objetivo

Crear un laboratorio local con backend, frontend, base de datos, Terraform, LocalStack, pipeline DEV y prompts corporativos gobernados para acelerar cambios técnicos y funcionales de forma segura.
